# Apiflow

Simple API testing. No bloat.

Postman is 200MB. Apiflow is a single `npm start`.

## Install

```bash
git clone https://github.com/quietdom/apiflow.git
cd apiflow
npm install
npm start
```

Opens at http://localhost:3456

## Features

- Send GET, POST, PUT, PATCH, DELETE requests
- Custom headers
- JSON body editor
- Bearer token / Basic auth
- Save requests in collections
- Response time, status code, size
- JSON syntax highlighting
- Dark theme

## Why

Postman requires an account. Insomnia wants your email. Bruno is 100MB.

Apiflow runs locally. No account. No telemetry. No bloat.

## Tech

- Express.js
- Vanilla HTML/CSS/JS
- No frameworks
